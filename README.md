# cmu-backend
Centro Medico Universitario (backend)
## Getting Started

Install dependency:

```bash
yarn install
```
Initialize database:

```bash
yarn init
```

## Run

Development:

```bash
yarn dev
```
open your browser in http://127.0.0.1:3000
Production:

```bash
yarn start
```

##Build
Create a build for production:

```bash
yarn build
```
